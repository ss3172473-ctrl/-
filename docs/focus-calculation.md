# 집중도 계산 방식

현재 집중도 계산방식은 **MediaPipe Pose 랜드마크**를 활용해서 집중도를 측정하고 있습니다.

---

## 분석 요소 (3가지)

| 요소 | 가중치 | 판단 기준 |
|------|--------|-----------|
| 고개 위치 | 40% | 코-어깨 거리가 0.15 미만이면 고개 숙임 |
| 시선 방향 | 40% | 코가 어깨 중심에서 벗어남, 귀 visibility 차이, 눈 높이 차이 |
| 존재 여부 | 20% | 코/눈 visibility가 낮으면 자리비움 |

---

## 세부 판단 기준

### 1. 고개 숙임 감지
- 코와 어깨 중심 사이 거리 < 0.15 → 고개 숙임

### 2. 옆 보기 감지 (OR 조건)
- 코가 어깨 중심에서 0.08 이상 벗어남
- 왼쪽/오른쪽 귀 visibility 차이 > 0.2
- 왼쪽/오른쪽 눈 높이 차이 > 0.03

### 3. 자세 불량
- 어깨 기울기 > 0.04 → 감점

### 4. 존재 여부
- 코 visibility < 0.5 또는 양쪽 눈 visibility < 0.3 → 자리비움

---

## 점수 계산 방식

1. **1초마다** 프레임 데이터 집계
2. 각 요소별 **100점 만점** 계산 후 가중 평균
3. 최종 점수: **0~100점**

---

## 집중도 레벨

| 레벨 | 점수 범위 |
|------|-----------|
| 🟢 집중 | 80점 이상 |
| 🟡 보통 | 50 ~ 79점 |
| 🟠 주의 | 30 ~ 49점 |
| 🔴 경고 | 0 ~ 29점 |
